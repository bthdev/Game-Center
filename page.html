Rolling up multiple rows into a single row and column with SQL Server

If you need a way to roll-up multiple rows into one row and one column. You know You can roll-up multiple rows into one row using Pivot, but you need all of the data concatenated into a single column in a single row. In this tip we look at a simple approach to accomplish this. And my solution is you should use STUFF function in Sql.

How to use ?

Code below:

SELECT STUFF((select ', ' + ColumnName FROM Your table name for xml path ('') ), 1, 2, '') AS ColumnName

How to find all table names with column name in Sql Server?

Please try the below query. Use sys.columns to get the details (For SQL SERVER 2008+) :

Code below:

SELECT c.name AS ColName, t.name AS TableName FROM sys.columns c JOIN sys.tables t ON c.object_id = t.object_id WHERE c.name LIKE '%YourColumn%';
